# 🛒 Supermarket Sales Analysis Dashboard (Excel)

## 📊 Overview

This project presents an interactive **Excel dashboard** for analyzing supermarket sales performance.
It provides insights into sales trends, profit distribution, product performance, and regional analysis using dynamic visuals and filters.

---

## 🚀 Key Features

* 📈 **Sales vs Discount Analysis** by product sub-category
* 🥧 **Products with Highest Discount Rate** (Pie Chart)
* 🌍 **Region-wise Sales Analysis** (West, East, Central, South)
* 💰 **Total Sales & Profit Percentage KPIs**
* 📅 **Day-wise and Month-wise Sales Trends**
* 🏙 **Top 10 Cities by Profit**
* 📦 **Sales Performance by Product Category**
* 🎯 Interactive filters (Year, Date, Category)
